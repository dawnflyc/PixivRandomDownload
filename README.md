# PixivRandomDownload

Pixiv图片随机下载。
Pixiv image random download.

可以用点赞收藏量进行筛选下载，比如:只下载点赞100以上的图。

也可以过滤18+图片，比如：都下、只下18+、不下18+

多线程下载

使用方法：
  命令行打开start.py
  参数1: 每个线程下载数  必须
  参数2: 开多少个线程    必须
  参数3: r18 只下载18+   可选 
  
  例如 pyton start.py 10 10       开10线程 每个线程下载10个有效图片 不过滤18+
       pyton start.py 10 10 r18   开10线程 每个线程下载10个有效图片 只要18+
